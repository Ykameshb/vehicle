# Vehicle Trajectory Prediction using LSTM

## Overview
This project predicts future vehicle GPS coordinates using an LSTM deep learning model.

## Dataset
Rome Taxi GPS Dataset (Kaggle)

## Features
- Data preprocessing
- Sequence creation
- LSTM neural network
- RMSE & MAE evaluation
- Future trajectory prediction
- Model saving

## Technologies Used
- Python
- TensorFlow
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Results
The model successfully predicts future GPS trajectory points with low error.
